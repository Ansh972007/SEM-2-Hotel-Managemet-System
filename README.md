# Hotel Management System

A **Hotel Management System** developed using **Java and PostgreSQL** to manage hotel operations such as room booking, customer management, and billing.  
This project was built as a semester project to demonstrate **backend application development and database integration using Java**.

---

## Features

• Room booking and reservation management  
• Customer information storage and management  
• Room availability tracking  
• Billing and payment record system  
• Database integration with PostgreSQL  
• Structured backend logic using Object-Oriented Programming

---

## Technologies Used

- Java
- PostgreSQL
- SQL
- XAMPP
- Object-Oriented Programming (OOP)

---

## Project Modules

### Room Management
Manage hotel room information and availability.

### Customer Management
Store and retrieve customer details.

### Booking System
Handle room reservations and booking records.

### Billing System
Generate and manage billing information for customers.

Before running the project please ensure the following:

• PostgreSQL database is installed and running  
• The database SQL file is imported  
• Database credentials in `DatabaseConnection.java` are configured correctly
